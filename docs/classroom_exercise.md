# 课堂实操练习：共被引边权重计算

## 1. 共被引边权重手算示例

### 1.1 示例数据

假设有以下论文引用关系：

| 引用论文 | 被引用论文 |
|---------|-----------|
| P1 | A, B, C |
| P2 | A, B, D |
| P3 | A, C, D |
| P4 | B, C, D |

### 1.2 共被引矩阵构建

| 论文 | A | B | C | D |
|------|---|---|---|---|
| A | - | 2 | 2 | 2 |
| B | 2 | - | 2 | 2 |
| C | 2 | 2 | - | 2 |
| D | 2 | 2 | 2 | - |

**解释**：
- A和B被P1和P2共同引用，共被引次数为2
- A和C被P1和P3共同引用，共被引次数为2
- A和D被P2和P3共同引用，共被引次数为2
- B和C被P1和P4共同引用，共被引次数为2
- B和D被P2和P4共同引用，共被引次数为2
- C和D被P3和P4共同引用，共被引次数为2

### 1.3 各论文被引用总次数

| 论文 | 被引用次数 |
|------|-----------|
| A | 3 (被P1, P2, P3引用) |
| B | 3 (被P1, P2, P4引用) |
| C | 3 (被P1, P3, P4引用) |
| D | 3 (被P2, P3, P4引用) |

## 2. Cosine相似度计算

### 2.1 公式

```
cosine_similarity(A, B) = (C(A,B)) / (sqrt(C(A)) * sqrt(C(B)))
```

### 2.2 计算示例

以A和B为例：

```
cosine_similarity(A, B) = 2 / (sqrt(3) * sqrt(3)) = 2 / 3 ≈ 0.6667
```

所有论文对的Cosine相似度：

| 论文对 | Cosine相似度 |
|--------|--------------|
| A-B | 2/3 ≈ 0.6667 |
| A-C | 2/3 ≈ 0.6667 |
| A-D | 2/3 ≈ 0.6667 |
| B-C | 2/3 ≈ 0.6667 |
| B-D | 2/3 ≈ 0.6667 |
| C-D | 2/3 ≈ 0.6667 |

## 3. Jaccard相似度计算

### 3.1 公式

```
jaccard_similarity(A, B) = (C(A,B)) / (C(A) + C(B) - C(A,B))
```

### 3.2 计算示例

以A和B为例：

```
jaccard_similarity(A, B) = 2 / (3 + 3 - 2) = 2 / 4 = 0.5
```

所有论文对的Jaccard相似度：

| 论文对 | Jaccard相似度 |
|--------|---------------|
| A-B | 2/4 = 0.5 |
| A-C | 2/4 = 0.5 |
| A-D | 2/4 = 0.5 |
| B-C | 2/4 = 0.5 |
| B-D | 2/4 = 0.5 |
| C-D | 2/4 = 0.5 |

## 4. 阈值选择逻辑

### 4.1 Top-N阈值

**选择逻辑**：按照权重或相似度从高到低排序，选择前N个边。

**示例**：
- 假设N=3，由于所有边的权重相同，任意选择3个边
- 假设N=6，选择所有边

### 4.2 Min Weight阈值

**选择逻辑**：设置一个阈值，选择权重或相似度大于等于该阈值的边。

**示例**：
- 阈值=1：选择所有边（权重都为2）
- 阈值=2：选择所有边（权重都为2）
- 阈值=3：不选择任何边

### 4.3 阈值敏感性分析

**实验设置**：
- 测试不同阈值对网络结构的影响
- 观察网络密度、节点数、边数的变化

**预期结果**：
- 随着阈值的增加，网络密度降低
- 高阈值会过滤掉弱连接，保留强连接
- 低阈值会包含更多连接，但可能引入噪声

## 5. 结论

1. **共被引边权重**：反映了两篇论文在学术网络中的关联强度
2. **Cosine相似度**：考虑了论文的被引用频率，更适合比较不同引用量级的论文
3. **Jaccard相似度**：关注共被引次数在总引用中的比例，更适合衡量相对关联度
4. **阈值选择**：
   - Top-N方法保证网络规模可控
   - Min Weight方法保留所有达到一定重要性的连接
   - 实际应用中应根据研究目的和数据特点选择合适的阈值策略
